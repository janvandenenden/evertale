## TEMPLATE PROMPTS

Below is the **full storyboard prompt set for the Momotaro book**, rewritten using a consistent generation structure.

Each prompt contains:

1. **Shot**
2. **Camera**
3. **Composition**
4. **Character blocking (pose + expression)**
5. **Environment**
6. **Placeholder rule (when protagonist appears)**
7. **Style block**

Reference sheets should be passed as **image inputs**, not described.

**Reference inputs**

- old_woman_character_sheet
- village_environment_sheet

**Prompt**

```
Wide establishing shot.

Composition: the river flows diagonally across the frame from lower left to upper right.
An elderly woman kneels at the riverbank on the right side of the frame washing clothes in a wooden tub.

Old Woman:
kneeling beside the water
hands working in the tub
expression calm and focused.
```
